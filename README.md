📚 Bookworm Library

Bookworm Library is a full-stack web application that allows users to browse, manage, and organize books in a digital library. Users can add books, update details, and manage their personal reading collection easily.

🚀 Features

- Browse available books

- Add new books & Edit book details

- Delete books & Search books by title or author

- User Authentication (Login/Register)



🛠 Tech Stack

Frontend: React.js,Tailwind CSS,JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Other Tools: Git & GitHub,Postman & JWT Authentication
